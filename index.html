Panduan Setup WiFi

Kipas Pintar ini belum dikenali oleh internet. Ikuti langkah cepat di bawah ini untuk menghubungkannya.

⚠️ SANGAT PENTING!

Saat memindahkan koneksi WiFi, JANGAN MENUTUP ATAU ME-REFRESH HALAMAN WEB INI. Biarkan berjalan di latar belakang.

Langkah-langkah:

  1. Buka Pengaturan WiFi di HP Anda.
  2. Hubungkan ke WiFi bernama Setup_Kipas_Pintar.
    (Jika HP Anda peringatkan 'Tidak ada Internet', pilih opsi 'Tetap Terhubung').
  3. Setelah WiFi berhasil tersambung, kembali buka halaman web ini (jangan di-*refresh*).
  4. Ketuk tombol merah di bawah untuk memasukkan sandi WiFi rumah asli Anda.
⚙️ Buka Pengaturan Setup

Apa selanjutnya?

Setelah menyimpan password, Kipas akan me-restart. Anda bisa mengeklik tanda silang (✕) di atas layar ini. Jika status Dashboard Anda masih Offline, artinya Kipas gagal terkoneksi WiFi, harap ulangi proses ini kembali.
