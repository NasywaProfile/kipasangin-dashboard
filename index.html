Panduan Setup WiFi

Kipas Pintar ini belum dikenali oleh internet. Ikuti 3 langkah cepat di bawah ini untuk menghubungkannya.

⚠️ SANGAT PENTING!

Selama melakukan proses perpindahan WiFi ini, mohon JANGAN MENUTUP ATAU ME-REFRESH HALAMAN WEB INI. Biarkan halaman web aplikasi ini tetap terbuka di latar belakang HP Anda.

Langkah-langkah:

  1. Masuk ke Pengaturan WiFi di Setting HP Anda.
  2. Cari dan Hubungkan ke WiFi bernama Setup_Kipas_Pintar.
    (Jika HP Anda tiba-tiba memberi peringatan 'Tidak ada Koneksi Internet', abaikan saja! Klik 'Tetap Terhubung' atau 'Keep WiFi').
  3. Setelah ikon WiFi di HP Anda menempel (tersambung), kembali buka halaman web ini (jangan di-refresh).
  4. Ketuk tombol merah besar di bawah ini untuk memasukkan nama & kata sandi WiFi rumah asli Anda.
⚙️ Buka Pengaturan (192.168.4.1)

Apa yang terjadi setelah tombol ditekan?

Browser akan memunculkan layar pengaturan Kipas. Silakan panggil dan simpan password rumah Anda di sana.

Memastikan Kipas Sudah Menyala (Berhasil):
Anda bisa mengeklik tanda silang (✕) di atas layar ini lalu menekan tombol Start Experience. Perhatikan status indikator Dashboard di pojok kiri layar web Anda. Jika statusnya masih bertuliskan Offline, artinya Kipas gagal terkoneksi WiFi Rumah, maka lakukan proses Repair/Setup di atas kembali dengan password yang benar.
